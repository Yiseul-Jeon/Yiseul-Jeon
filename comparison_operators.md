### ▶ 오류의 세 가지 종류

| 유형 | 설명 |
| --- | --- |
| Syntax Error | 문법 오류 (오타, 괄호 누락 등) |
| Logic Error | 실행은 되지만 의도한 결과가 나오지 않음 |
| Runtime Error | 실행 도중 발생하는 오류 (예: 0으로 나누기 등) |

> 예외 처리는 try ~ except ~ finally 구조로 처리
